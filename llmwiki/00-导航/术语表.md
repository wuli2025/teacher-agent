# 术语表

> RAG / 检索 / Agent 高频中英术语。检索高频词建议中英并列。深入见各专页。

## 检索基础
- **RAG (Retrieval-Augmented Generation, 检索增强生成)** —— 先检索相关资料再让模型基于它生成答案。
- **Embedding (嵌入 / 向量)** —— 把文本编码成稠密向量，语义近的向量也近。见 [[向量检索]]。
- **ANN (Approximate Nearest Neighbor, 近似最近邻)** —— 大规模向量快速找近邻的算法，如 HNSW/IVF/PQ。
- **BM25** —— 经典词频-逆文档频率打分的全文检索算法，字面精准。见 [[BM25与全文检索]]。
- **Hybrid Search (混合检索)** —— 稠密(向量)+稀疏(BM25)并用。见 [[混合检索与RRF]]。
- **RRF (Reciprocal Rank Fusion, 倒数排名融合)** —— 只按排名位置合并多路结果，常数 k≈60。
- **Rerank (重排)** —— 对粗召回结果用更准的模型重新排序。见 [[重排rerank]]。
- **Cross-encoder** —— 把 query 和文档一起送入模型打分，准但慢；用于重排。
- **ColBERT / 后期交互 (late interaction)** —— token 级多向量 + MaxSim，介于 bi-encoder 和 cross-encoder 之间。
- **Chunking (切块/分块)** —— 把文档切成检索单元。见 [[分块策略]]。
- **MTEB / C-MTEB** —— (中文)嵌入模型评测基准。见 [[嵌入模型选型]]。
- **Matryoshka (套娃嵌入)** —— 向量前若干维即可截断使用，省存储。

## 查询侧
- **Query Rewrite (查询改写)** / **Query Expansion (查询扩展)** —— 检索前优化/扩写问题。见 [[查询改写]]。
- **HyDE** —— 先让模型写假设答案，用它的向量去检索。见 [[HyDE]]。
- **Step-back prompting** —— 先退一步问更抽象的问题。
- **Multi-query / 子问题分解** —— 把复杂问题拆成多个子问。

## 高级架构
- **GraphRAG** —— 把语料抽成实体关系图+社区摘要再检索。见 [[GraphRAG]]。
- **Agentic RAG (智能体检索)** —— 让模型用工具自己多轮检索。见 [[Agentic-RAG]]。
- **Self-RAG** —— 自反思检索。见 [[Self-RAG]]。
- **CRAG (Corrective RAG, 纠错式)** —— 评估检索质量并兜底。见 [[CRAG]]。
- **RAPTOR** —— 递归摘要树。见 [[RAPTOR]]。
- **Contextual Retrieval (上下文检索)** —— Anthropic：给每块加情境。见 [[上下文检索-Contextual-Retrieval]]。
- **Late Chunking (后期分块)** —— Jina：先整篇编码再切。见 [[Late-Chunking]]。

## Agent 与上下文
- **Context Engineering (上下文工程)** —— 策划维护送进窗口的最优 token 集。见 [[上下文工程]]。
- **Attention budget (注意力预算)** —— 上下文塞太多会退化。
- **Lost in the middle** —— 长上下文中间信息易被忽略。见 [[长上下文vs RAG之争]]。
- **Context rot (上下文腐烂)** —— 输入越长性能越退化。
- **KV-Cache** —— 缓存注意力键值，缓存命中率决定 agent 成本。
- **ReAct** —— Reason+Act 交替的 agent 动作循环。
- **Memory layer (记忆层)** —— 如 Mem0/Letta，给 agent 跨会话记忆。见 [[Agent记忆系统]]。

## 评测
- **Recall@k / Precision@k / NDCG / MRR** —— 检索指标。见 [[RAG评测方法]]。
- **Faithfulness (忠实度/groundedness)** —— 答案是否被检索内容支撑（防幻觉）。
- **RAGAS** —— 主流 RAG 评测框架。见 [[RAGAS]]。
- **LLM-as-a-judge** —— 用 LLM 当评委打分，注意位置/长度/自偏好等偏差。
