# 评分卡（foundation/rubric.md）

> 两个用途：① 采集阶段给候选案例打分（≥28 分才值得解构成证据卡）；② 实战校验阶段给人格产出的样张打分（决定升级是否落盘）。
> 修改权限：仅人类。版本 v1.0（2026-07-04）。

## 一票否决项（任意一条命中 → 0 分，直接不合格）

- [ ] 存在深字压深底 / 浅字压浅底（含 color-mix 染底后的两端）[A1]
- [ ] 存在元素重叠压字、内容溢出截断 [A3]
- [ ] 灰度检查：转灰后无法分辨主角与背景信息 [A5/A2]
- [ ] （样张）手写死文字颜色绕过 token [A11]

## 七维评分（每维 1–5 分，总分 35）

| # | 维度 | 5 分锚点 | 3 分锚点 | 1 分锚点 |
|---|------|---------|---------|---------|
| R1 | 对比与可读 [A1] | 任何距离眯眼可读，小字依旧清晰 | 主体清晰，个别小字偏弱 | 需要凑近辨认 |
| R2 | 层级与焦点 [A2/A8] | 3 秒内动线清晰，每页一个明确主角 | 能找到重点但需扫两遍 | 满页平权，不知看哪 |
| R3 | 色彩和谐 [A5] | 色相模型明确，60-30-10 成立，灰度后层级依旧 | 和谐但保守/寡淡，或强调色略泛滥 | 色相打架或全灰无生气 |
| R4 | 排版工艺 [A4/A6] | 网格严整、字阶成模数、细节（行长/行高/中西文间隙）讲究 | 大体整齐，个别间距无政府 | 明显未对齐、字阶随意 |
| R5 | 留白与节奏 [A3/A7] | 疏密有编曲感，翻页强弱交替 | 密度均匀无节奏 | 拥挤或空得失控 |
| R6 | 一致性与克制 [A7/A10/A11] | 细节全局同语言，无一多余元素 | 偶有风格漂移或冗余装饰 | 页页各说各话 |
| R7 | 记忆点与得体 [A12] | 合上后能想起画面，且气质与内容严丝合缝 | 得体但无记忆点 | 风格与内容错位 |

## 分数线

- **采集侧**：案例 ≥28/35 → 值得写证据卡；≥32 → 标 `exemplar`（示范级，人格提炼优先引用）。
- **实战校验侧**（人格升级后的样张）：
  - 总分 ≥28 且 无否决项 且 不低于该人格上一版样张总分 → 升级落盘 ✅
  - 低于上一版 ≥3 分 → 回滚本轮全部 delta，journal 记失败原因 ❌
  - 其余 → 落盘但在人格文件版本志标 `provisional`，下一轮复验

## 评分纪律

- 每一维必须写一句**证据句**（指着具体页面/元素说话），禁止裸打分。
- 给自己产出打分时，先休息性换位：假装这是竞争对手的作品再评。
- 同一样张如条件允许由两次独立评分（不同会话/不同视角提示）取均值。
