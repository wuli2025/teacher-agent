**平台：微信公众号 · 投递流程**

**登录态**：沿用 `~/.polaris-mp-profile`（向后兼容），未登录交账号管家扫码。

**投递范式**：调用现有 wechat_yiban.py 通道，走 ProseMirror 合成粘贴（ClipboardEvent + DataTransfer），降级 execCommand/innerHTML。填充图文编辑器标题、作者、正文（微信兼容 HTML），上传封面 2.35:1。

**流程步骤**：打开 mp.weixin.qq.com 图文编辑 → 填标题/正文/封面 → 触发「保存为草稿」→ 校验草稿箱出现该条 → 返回 `draft_uploaded`。

**平台红线（铁律）**：只保存草稿，绝不点「群发/发表/定时群发」。素材库有每日草稿数与群发次数限制，投递只碰草稿不碰群发。

**失败降级**：适配失败或未登录 → 打开编辑页 + 正文进剪贴板 + 提示「已打开公众号编辑页，正文在剪贴板，请手动粘贴并自行确认发布」，浏览器保持打开。
