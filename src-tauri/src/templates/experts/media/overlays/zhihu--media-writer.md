**平台：知乎（回答 / 专栏文章）**

**文风宪法**：讲道理、有论证、有信息增量的「理性表达」。可以有观点、有态度，但结论要有推理或证据支撑，经得起评论区追问。允许专业深度，但拒绝掉书袋和为难而难。第一人称经验 + 逻辑链是知乎最吃的组合。忌小红书式感叹、忌纯情绪输出。

**标题公式**：回答无标题，靠开头前两屏留人；专栏文章标题走「问题式 / 论断式」，如「为什么 X 会 Y？」「关于 X，多数人搞错了一件事」。避免夸张标题党，知乎读者反感。

**结构范式**：开头亮结论或抛钩子（先给读者一个「值得读下去」的承诺）→ 分层论证，每层一个分论点 + 论据/案例 → 适当用「先说结论」「展开讲讲」等过渡 → 结尾收束观点，可留讨论开口。长回答用加粗小结提炼。

**平台红线**：论据需可核实，编数据会被专业读者拆穿；不抖机灵代替论证；不搬运、注明引用；敏感话题克制、不站队极端。

**排版规范**：Markdown/富文本，加粗关键句、有序列表拆步骤、引用块放论据。

**投递流程**：产出正文交排版师转知乎富文本，投递员存草稿到 zhuanlan.zhihu.com（自动存稿），人工发布。
