| 代号 | 目标 | 指标 |
|---|---|---|
| **G-1** | 一条龙跑通：人设 → 竖版叙事视频草稿 | ≤ 10 分钟，零手写代码 |

## 4. 信息架构

左栏三模块 + 设置；**语音输入**与**生图**是贯穿能力，不占一级导航却随处可唤起。

```
┌──────────────┐    左栏第一项 = 视频；进入即停在「生成剧本」步
│  ▍视频        │◀── 人生副本 5 步工作流
│   PPT         │
│   对话        │
│  ────────     │
│   设置        │◀── API / 生图 / 语音 / 参考库
└──────────────┘
   ↑ 底部常驻：长按说话（语音输入，全局）
```

- **模块一 · 视频**：本 PRD 主战场，§5。
- **模块二 · PPT**：复用 `DeckStudio`，仅对齐主视觉。
- **模块三 · 对话**：复用 `ChatPanel`，作「随手改剧本」兜底。
- **能力 · 语音输入**：§6，全局浮层。
- **能力 · 生图**：§7，既嵌在视频第 4 步，也可单开「画面工作室」。

> 落地：`stores/app.ts` 的 `ViewKey` 收敛为 `'video' | 'deck' | 'chat' | 'settings'`，其余 view 代码保留但从主导航隐藏。

## 5. 视频模块（人生副本 5 步）

5 步横向 Stepper，对应原工作流：

```
┌─────────────────────────────────────────────────────────────────────┐
│  ●━━━━━━●━━━━━━○──────○──────○        墨蓝填充=已完成 水墨描边=当前     │
│  1人设   2剧本   3配音   4画面   5成片                                  │
├──────────────┬──────────────────────────────────────────────────────┤
│  左：操作区   │  右：深色电影感预览区（竖版 9:16 实时预览）            │
│  浅底克制     │                                                       │
└──────────────┴──────────────────────────────────────────────────────┘
```

### C-5.1 步骤一 · 人设选题
| 任务 | 验收 | 落点 |
|---|---|---|
| T-5.1.1 人设卡：主题 + 一句话冲突点 | 自动生成标题「今天体验 XX 的一生」 | 新增 `PersonaCard` 区块 |
| T-5.1.2 灵感库 16 个高冲突人设（偷车贼/杀猪匠/单身一生/外卖骑手/守墓人/赌徒/钉子户/网瘾少年/凤凰男/拆迁户/北漂/相亲达人/夜班护士/退伍兵/收废品/直播主播） | 点一下填入 | 新增 `lib/lifePersonas.ts` |
| T-5.1.3 职业点网状图（增强） | 点节点=选人设 | 复用已装的 `cytoscape` |

### C-5.2 步骤二 · 生成剧本（节奏引擎）
左栏点「视频」默认就停在这一步——「生成剧本」即第一动作。

| 任务 | 验收 | 落点 |
|---|---|---|
| T-5.2.1 6 级分层剧本：第一人称，每级 100–150 字，含 场景/动作/心理/字幕 | 返回结构化 `levels[]`，字段齐全 | 新增 `lib/lifeScript.ts` |
| T-5.2.2 **节奏模式库**（≥5 套，可套用） | 切节奏，级数/张力曲线/转折位实时变 | 新增 `lib/rhythmModes.ts` |
| T-5.2.3 提示词模板可见可改可重置 | 内置默认模板 | `lifeScript.ts` 内 `PROMPT_TEMPLATE` |
| T-5.2.4 逐级可编辑、锁定进下一步 | 编辑持久化到草稿 | `ScriptEditor` 区块 |

**节奏模式库**：每套 = 张力曲线 + 级数 + 转折点 + 结尾处理

| 模式 | 级数 | 张力曲线 | 适用 |
|---|---|---|---|
| 平步沉沦（默认） | 6 | 缓起→逐级加压→第5级崩塌→第6级余味 | 偷车贼/赌徒「下坠人生」 |
| 高开低走 | 6 | 第1级巅峰→持续滑落→结尾顿悟 | 单身一生/中年危机 |
| 谷底反弹 | 7 | 低开→第4级触底→后段逆袭 | 励志翻身 |
| 平行抉择 | 5+分支 | 第3级岔路→双结局 | 多集系列 |
| 快剪高燃 | 4 | 短促、每级强钩子，≤80字 | 卡点快节奏 |

```ts
interface RhythmMode {
  id: string; name: string; levels: number;
  curve: number[];               // 每级张力 0-100，驱动配乐/画面冷暖
  twistAt: number;               // 转折级序号
  branchAt?: number;             // 分支级（可选）
  wordPerLevel: [number, number];
  promptHint: string;            // 拼进 LLM 的节奏指令
}
```

### C-5.3 步骤三 · 叙事配音（TTS）
| 任务 | 验收 | 落点 |
|---|---|---|
| T-5.3.1 逐级生成低沉叙事音，可试听/重生 | 每级一段音频 | 复用语音供应商 |

| T-5.3.3 环境音叠加（雨声/键盘/低频氛围），音量可调 | 可选叠加 | 本地静音素材 |

### C-5.4 步骤四 · 画面生成
直接调用**生图模块**（§7），逐级配图，画风统一。

| 任务 | 验收 | 落点 |

| T-5.4.1 逐级配图：每级 → 1 张竖版图，全片画风一致（拼 `storyStyles` 后缀） | 画风统一 | 复用 VideoCourseStudio 生图链路 |

| T-5.4.3 张力驱动构图：高张力级自动偏暗/特写 | 张力映射 prompt | 用节奏 `curve[]` |

### C-5.5 步骤五 · 合成导出
| 任务 | 验收 | 落点 |
|---|---|---|

## 6. 语音输入模块（豆包式）

> 沿用既定设计：**长按录音 + 流式 partial 上屏 + 松手定稿**，走云端流式 ASR（火山首选），不上本地 whisper。

```
                          ┌───────────────────────────┐
   底部常驻按钮            │  我想体验一个偷车贼的一    │◀ partial 实时上屏（灰字）
   ●  长按说话    ───按住─►│  生，从第一次撬锁到…       │
                          └───────────────────────────┘
   松手 → 定稿（黑字）→ 自动填入当前输入框（人设/剧本/对话）
```

| 任务 | 验收 | 落点 |
|---|---|---|
| T-6.1 长按唤起录音浮层，松手定稿 | 按住录、松手停 | 复用 `VoiceOverlay.vue` |
| T-6.2 流式 partial 上屏，首屏延迟 < 300ms | 边说边出灰字 | 火山流式 ASR WebSocket |
| T-6.3 定稿自动注入当前焦点输入框 | 人设/剧本/对话三处通用 | 全局 composable `useVoiceInput` |
| T-6.4 ASR 供应商可配（火山默认，可换） | 设置 → 语音 Tab | 扩展 `VoiceSettings.vue` |

## 9. UI / 视觉规范（高级感）

- **主色** 墨蓝 `#2c4661`；**选中态** 淡灰 `#ebebe8`；**朱砂** 仅警告/高张力强调。
- **绝不用 emoji**；图标用 `@lucide/vue` + 自绘水墨线条。
- **布局母版**：左浅底操作区（克制、留白）＋右深色电影感预览区（竖版黑场聚焦）。
- **Stepper**：已完成段墨蓝实心，当前段水墨描边脉冲动画。
- **节奏曲线**：手绘感折线可视化 `curve[]`，高张力偏朱砂、低张力偏墨蓝。
- **语音浮层**：底部常驻胶囊「长按说话」，按下水墨涟漪扩散，partial 灰字、定稿黑字。
- **生图网格**：竖版卡片，hover 浮起 + 重生/锁定角标，整体哑光质感。
- **气质关键词**：留白、克制、电影感、东方水墨——对标「高级而不喧哗」。

## 10. 端到端数据流

```
       ┌─ 语音输入(可选) ─┐
人设卡 ◀┘                 └► persona
   │ + RhythmMode
   ▼
LLM(lifeScript) ──► levels[]{scene,action,psych,caption,tension}
   ├──► TTS(narrator+环境音) ──► audio[level]
   ├──► 生图(style + tension) ──► image[level]
   ▼
Timeline 合成 ──► 导出包(imgs + audio + srt + 剪映清单) + 发布文案
```

草稿 `StoryProject`（JSON）持久化到工作目录，随时续编。
