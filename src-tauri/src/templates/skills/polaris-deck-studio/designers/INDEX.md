# 设计师人格总索引（designers/INDEX.md）

> 演示工坊 / 网站生成的「选设计师」入口。每位设计师是一份 `<id>.md` 人格文件（含宣言、拨盘、色板、字阶、版式、禁忌、实现映射）。
> **动手前**：① 读本索引挑一位（用户指定则用指定的；`auto` 则按下方路由表挑）→ ② 读该设计师的 `<id>.md` 全文 → ③ 按 `_foundation/taste.md` T2 输出一行设计判读（含拨盘值）→ ④ 严格照它的第 3–10 节做，全程守住 `_foundation/aesthetics.md` 的 12 条美学公理、`_foundation/taste.md` 的 10 条工艺纪律与 `design.md` 第 0 条对比度铁律。
> 每位设计师有「拿手三套系」（同风格三个可比较变体），用户可再挑一套；没挑就用该设计师第一套（标准款）。
> 每位设计师 frontmatter 带默认拨盘 `dials`（V 变化度 / M 动效度 / D 密度，1–10，定义见 taste.md T1），按内容气质可各 ±1。

## auto 路由表（用户没指定设计师时，按内容气质自动请人）

先判断内容的**场合 + 受众 + 情绪温度**，命中即选：

- 向上汇报 / 周报月报 / 经营分析 / 竞标 / 答辩 → **瑞士格大师**（`swiss-modernist`，严谨网格、结论先行）
- 党建 / 政务 / 国企 / 表彰大会 → **瑞士格大师**（`swiss-modernist`，庄重靠对齐排出来）
- 产品发布 / 融资路演 / 科技品牌 / 高端商务 / 未明确但想「高级炫」→ **发布会大师**（`keynote-tech`）
- SaaS/AI 官网气质、要「干净高级」→ **弥散光大师**（`mist-gradient`）
- 功能点多、要一图打尽 / 年终总结 → **便当格大师**（`bento-grid`）
- 开发者 / 技术 / 数据面板 / 极客 → **玻璃酥大师**（`glass-crisp`）
- 课程 / 培训 / 教学 → **课件大师**（`pedagogy-clarity`）；中小学/亲子 → **手绘涂鸦大师** 或 **粘土大师**
- 知识分享 / 生活方式 / 种草 / 消费品 / 社媒 / 女性向 → **小红书大师**（`xhs-life`）
- 年报 / 战略 / 学术 / 沉静品牌 / 理念致辞，要极稳极专业 → **瑞士格大师**（`swiss-modernist`）
- 中国风 / 文旅 / 国货 / 奢侈华彩 / 年会颁奖盛典 → **国潮彩大师**（`oriental-grandeur`）
- 儿童 / 创意 / 招聘 / 想被转发的活力向 → **波普糖大师**（`memphis-pop`）
- 教育 / 亲子 / 健康软萌 → **粘土大师**（`clay-soft`）

**兜底**：判断不了就用 **发布会大师**（`keynote-tech`）——它是默认高级款，深空 + 相位受控渐变 + 巨数页，一眼像一线发布会。

## 硬约束（所有设计师之上）
1. `_foundation/aesthetics.md` 12 公理 + `design.md` 第 0 条对比度铁律，任何设计师都不得违反（深底浅字/浅底深字，小字尤甚，只用 token 不写死颜色）。
1.5 **语言收敛、结构发散**（taste.md T11）：圆角/投影/图标/缓动全篇各一套；版式结构必须变化——「全居中垂直堆叠 + 每区眉标」和 T4 的一二代 AI 痕迹（图标芯片矩阵、彩色竖条卡、幽灵序号、Tailwind 靛蓝…）一律封死。
2. 选定设计师后**照它的禁忌清单办**——每位的「绝不」比通用规则更狠，那是它的性格。
3. 版式必须混排（一份 10 页 ≥4–5 种页型），拒绝通篇「标题+项目符号」。
4. **动手前**按 `_foundation/taste.md` 设定三拨盘（设计师默认值 ±1）并输出一行设计判读（T2 格式）——先声明，再动手。
5. **交付前**跑 `_foundation/taste.md` 的 Pre-Flight 清单（眉标数、版式家族数、AI 痕迹、动效理由……全部可数可判），任何一条打不了勾即返工；产物首行写遥测锚点注释 `<!-- designer: <id> · dials V/M/D · preflight n/n -->`。
